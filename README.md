# IP Messenger V6

IP Messenger V6 is a fast, lightweight LAN (Local Area Network) messaging 
application for Windows. Built with Electron and React, it allows users 
on the same network to instantly send messages and files — completely 
offline, with no server or internet connection required.

## ✨ Features
- 📡 Instant LAN-based messaging — no internet needed
- 📁 File sharing over local network
- 🔔 Native desktop notifications with unread badge counts
- 🖥️ Clean, modern UI built with React
- ⚡ Lightweight and fast — runs quietly in the system tray
- 🔄 Auto-update support via GitHub Releases

## 🛠️ Built With
- Electron
- React + Vite
- NSIS (Windows Installer)

## 📦 Installation
Download the latest `IP_Messager_Setup.exe` from the Releases page and run it.
No configuration needed — just install and start chatting on your network.
